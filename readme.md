# Laravel Reflection

Generate database
```
php artisan migrate:refresh --seed
```

Host locally
```
php artisan serve
```

Access at http://localhost:8000/

## SQlite CLI

powershell:
```
& 'C:\Program Files\sqlite3\sqlite3.exe' database\database.sqlite
```

Make the output more human-readable:
```
.mode column
.headers on
```
